# Icons

Icons system is based on feather icons imported from 'react-icons' library.

## Usage

Import an icon from 'react-icons' and use as normal react component.

`import { IconName } from "react-icons/fi";`

```
<IconName />
```

**Visit react-icons documentation to see list of icons available. [Documentation](https://react-icons.github.io/react-icons/icons?name=fi).**
